## hadoop vs spark

| Feature    | Hadoop MapReduce             | Spark                                   |
| ---------- | ---------------------------- | --------------------------------------- |
| Processing | Disk-based                   | In-memory                               |
| Speed      | Slower                       | 10-100x faster                          |
| API        | Low-level                    | High-level (RDD, DataFrame, SQL, MLlib) |
| Use Cases  | Simple ETL, batch processing | ETL, ML, streaming                      |
